Proteus

Proteus es un software profesional utilizado para diseñar, simular y probar circuitos electrónicos de manera virtual antes de armarlos físicamente. Este programa permite crear esquemas detallados con resistencias, sensores, microcontroladores y otros componentes, y ofrece la posibilidad de ver cómo funciona el circuito en tiempo real.

Circuito en serie

Este circuito es simple de controlar un LED utilizando una fuente de alimentación de 5 V, un interruptor, una resistencia y un LED, cuando el interruptor está cerrado, la corriente fluye desde la fuente de 5 V, pasa por la resistencia y llega al LED, haciendo que se encienda.

Medicion de temperatura y humedad

Es un sistema con Arduino Uno que mide temperatura y humedad (DHT22), distancia (sensor ultrasónico HC-SR04) y nivel de luz (LDR), y muestra los datos en una pantalla LCD.

Leds

Es un sistema con Arduino Uno que controla cuatro LEDs de colores mediante dos potenciómetros y dos interruptores.
